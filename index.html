Welcome to the COVID Testing Site Locator!

This app allows you to find available COVID Testing Sites near you with the click of a single button. Clicking the button labeled "Find A Location Near Me" will render a list of locations under the box labeled "Locations Near You" and generate a map inside the gray box, starting with the nearest location. Clicking through the "Select" buttons in the locations list will render a new map for the selected location. Under the map, a "Visit Selected" button is available that allows you to store locations in your device's local storage and render them inside of the box labeled "Visited Locations", so you can keep track of which locations you've visited if any. Feel free to clear out the visited locations list at any point using the "Clear Visited" button.
