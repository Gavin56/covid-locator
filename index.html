Welcome to the COVID Testing Site Locator!

This app allows you to find available COVID Testing Sites near you with the click of a single button. Start by clicking the button labeled "Find A Location Near Me" to render a list of locations under the box labeled "Locations Near You" and generate a map inside the gray box, starting with the nearest location. Choose another location by clicking the "Select" button next to the prefered testing site and display a new map for the selected location. Under the map, a "Visit Selected" button is available that preserves the date and address of the selected testing site. This will be displayed in the "Visited Locations" section. Feel free to clear out the visited locations list at any point using the "Clear Visited" button. Lastly, there is a drop-down menu on the top right that contains a link to the CDC Guidelines website and our App Instructions if you ever need to refresh your memory.
